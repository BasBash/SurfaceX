# Componentes de Hardware

- **Câmera**: captura imagens para reconstrução 3D.
- **Laser**: projeta uma linha de referência na cena.
- **Mesa**: gira o objeto para escaneamento completo.
- **Marcadores ArUco**: usados para rastreamento e calibração.
- **Placa Charuco**: fixada na mesa para calibração precisa.
