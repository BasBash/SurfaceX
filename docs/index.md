# Scanner 3D MITEC Surface

Bem-vindo à documentação oficial do projeto **MITEC Surface**.

Este projeto combina **hardware** (câmera, laser, Arduino, mesa motorizada, marcadores ArUco e placa Charuco) e **software** (interface gráfica para controle e processamento) para criar um sistema de escaneamento 3D acessível e modular.

Use o menu lateral para navegar entre as seções.
